# Coffee-Sales-Analysis-Dashboard
☕ **Coffee Shop Sales Analysis Dashboard**

📌 **Overview**-
This project presents a comprehensive analysis of coffee shop sales data using an interactive Excel dashboard. It helps uncover key business insights such as sales trends, customer behavior, and product performance.

🎯 **Objectives**-
Analyze sales performance across different time periods
Identify best-selling products and categories
Track revenue and order trends
Provide actionable insights for business growth

🛠️ **Tools & Technologies**-
Microsoft Excel (Dashboard, Pivot Tables, Charts)
Data Cleaning & Transformation (Excel Functions)

📂 **Dataset**-
The dataset includes:
Order ID and transaction details
Product names and categories
Quantity and sales amount
Date and time of purchase

📈 **Key Features**-
📊 Interactive dashboard with slicers and filters
💰 KPI metrics (Total Revenue, Total Orders, Quantity Sold)
📅 Time-based analysis (Daily, Monthly trends)
🥐 Category-wise and product-wise performance
⏰ Peak sales hours analysis

🔍 **Key Insights**-
Certain products (like coffee & bakery items) contribute the highest revenue
Sales peak during specific hours of the day (morning/evening rush)
Weekends show higher sales compared to weekdays
A few top products drive a major portion of total sales

🚀 **How to Use**-
Download the Excel file from the repository
Open the dashboard in Microsoft Excel
Use slicers to filter data (date, category, etc.)
Explore charts to gain insights
